# 사용 가이드

## 🚀 빠른 시작

### 1. 서버 실행

```bash
bun dev
```

브라우저에서 `http://localhost:3000`으로 접속합니다.

### 2. 호스트 설정 (리그오브레전드 클라이언트가 있는 컴퓨터)

1. **리그오브레전드 클라이언트를 실행**합니다
2. 웹 페이지에서 **"방 만들기"** 버튼을 클릭합니다
3. 생성된 **방 코드** (예: NWGRTT)를 복사합니다
4. 방 코드를 친구들에게 공유합니다
5. **"리그오브레전드 클라이언트 연결"** 버튼을 클릭합니다
6. 챔피언 선택 단계에 진입하면 자동으로 데이터가 공유됩니다

### 3. 게스트 설정 (관전하는 사람)

1. 웹 페이지에 접속합니다
2. 공유받은 **방 코드**를 입력합니다
3. **"참가"** 버튼을 클릭합니다
4. 호스트의 밴픽 화면이 실시간으로 표시됩니다

## 📊 화면 설명

### 호스트 화면

- **방 코드**: 다른 사람들이 접속할 수 있는 고유 코드
- **리그오브레전드 클라이언트 상태**: 연결 여부 표시
- **연결된 사용자**: 현재 방에 접속한 사람 수
- **밴픽 데이터**: 실시간 챔피언 선택 정보

### 게스트 화면

- **방 코드 입력**: 호스트에게 받은 코드 입력
- **밴픽 데이터**: 호스트의 밴픽 화면 실시간 표시

## 🔧 문제 해결

### "서버 연결에 실패했습니다"

- 서버가 제대로 실행되고 있는지 확인하세요
- 브라우저를 새로고침해보세요
- 방화벽 설정을 확인하세요

### "리그오브레전드 클라이언트를 찾을 수 없습니다"

- 리그오브레전드 클라이언트가 실행 중인지 확인하세요
- 클라이언트를 재시작해보세요
- 관리자 권한으로 실행해보세요

### P2P 연결이 안 됩니다

- 양쪽 모두 방화벽 설정을 확인하세요
- 다른 방 코드로 다시 시도해보세요
- 브라우저를 새로고침하고 다시 시도하세요

## 💡 팁

- 호스트는 게임을 시작하기 전에 미리 방을 만들어두는 것이 좋습니다
- 방 코드는 대소문자를 구분하지 않습니다
- 여러 명이 동시에 관전할 수 있습니다
- 챔피언 선택이 끝나면 데이터가 자동으로 초기화됩니다

## 🌐 네트워크 설정

### 로컬 네트워크에서 사용

같은 네트워크(Wi-Fi)에 있는 경우:

1. 호스트 컴퓨터의 IP 주소를 확인합니다
2. 게스트는 `http://[호스트IP]:3000`으로 접속합니다

### 인터넷을 통한 공유

외부에서 접속하려면:

1. 포트 포워딩 설정 (포트 3000)
2. 호스트의 공인 IP 주소 확인
3. 게스트는 `http://[공인IP]:3000`으로 접속합니다

또는 ngrok 같은 터널링 서비스를 사용할 수 있습니다:

```bash
ngrok http 3000
```
