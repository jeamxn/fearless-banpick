# 🎮 리그오브레전드 밴픽 공유

리그오브레전드 챔피언 선택(밴픽) 화면을 실시간으로 공유할 수 있는 웹 애플리케이션입니다.

## ✨ 주요 기능

- 🏠 **호스트 모드**: 리그 클라이언트가 실행 중인 컴퓨터에서 방을 만들고 밴픽 데이터를 실시간으로 공유
- 👥 **게스트 모드**: 방 코드를 입력하여 다른 사람의 밴픽을 실시간으로 관전
- 🔄 **P2P 연결**: PeerJS를 사용한 서버리스 P2P 데이터 전송
- ⚡ **실시간 업데이트**: WebSocket을 통한 즉각적인 데이터 동기화

## 🚀 시작하기

### 설치

```bash
bun install
```

### 개발 서버 실행

```bash
bun dev
```

서버가 시작되면 브라우저에서 `http://localhost:3000`으로 접속하세요.

### 프로덕션 빌드

```bash
bun start
```

## 📖 사용 방법

### 1. 호스트 (리그 클라이언트 실행 중인 사람)

1. 리그오브레전드 클라이언트를 실행합니다
2. 웹 브라우저에서 애플리케이션에 접속합니다
3. **"방 만들기"** 버튼을 클릭합니다
4. 생성된 **방 코드**를 친구들에게 공유합니다
5. 챔피언 선택 단계에 진입하면 자동으로 데이터가 공유됩니다

### 2. 게스트 (관전하는 사람)

1. 웹 브라우저에서 애플리케이션에 접속합니다
2. 공유받은 **방 코드**를 입력합니다
3. **"참가"** 버튼을 클릭합니다
4. 호스트의 밴픽 화면이 실시간으로 표시됩니다

## 🛠️ 기술 스택

- **런타임**: [Bun](https://bun.sh)
- **프레임워크**: React 19
- **스타일링**: Tailwind CSS + shadcn/ui
- **P2P**: PeerJS
- **리그 연동**: league-connect

## 📝 주의사항

- 호스트는 리그오브레전드 클라이언트가 실행 중이어야 합니다
- 방을 만든 호스트의 컴퓨터에서 서버가 실행되어야 합니다
- P2P 연결을 위해 방화벽 설정이 필요할 수 있습니다

## 🔧 개발

이 프로젝트는 Bun을 사용하여 개발되었습니다. 

- `bun dev`: 개발 서버 실행 (HMR 활성화)
- `bun start`: 프로덕션 모드로 실행
- `bun build`: 프로덕션 빌드
