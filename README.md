# 🎮 리그오브레전드 밴픽 공유

리그오브레전드 챔피언 선택(밴픽) 화면을 실시간으로 공유할 수 있는 웹 애플리케이션입니다.

> 🤖 **이 프로젝트는 Claude Sonnet 4.5를 사용하여 개발되었습니다.**  
> 모든 코드, 디자인, 아키텍처가 AI에 의해 생성되었습니다.

## ✨ 주요 기능

- 🏠 **호스트 모드**: 리그오브레전드 클라이언트가 실행 중인 컴퓨터에서 방을 만들고 밴픽 데이터를 실시간으로 공유
- 👥 **게스트 모드**: 방 코드를 입력하여 다른 사람의 밴픽을 실시간으로 관전
- 🔄 **P2P 연결**: PeerJS를 사용한 서버리스 P2P 데이터 전송
- ⚡ **실시간 업데이트**: WebSocket을 통한 즉각적인 데이터 동기화
- ⚔️ **피어리스 제도**: 논/소프트/하드 피어리스 모드 지원
- 📊 **세트 기록**: 모든 게임 세트의 밴픽 내역 자동 저장 및 표시
- 🎨 **챔피언 초상화**: Data Dragon API를 통한 실시간 챔피언 이미지 표시

## 🚀 시작하기

### 설치

```bash
bun install
```

### 개발 서버 실행

```bash
bun dev
```

서버가 시작되면 브라우저에서 `http://localhost:3000`으로 접속하세요.

### 프로덕션 빌드

```bash
bun start
```

## 📖 사용 방법

### 1. 호스트 (리그오브레전드 클라이언트 실행 중인 사람)

1. 리그오브레전드 클라이언트를 실행합니다
2. 웹 브라우저에서 애플리케이션에 접속합니다
3. **"방 만들기"** 버튼을 클릭합니다
4. 생성된 **방 코드**를 친구들에게 공유합니다
5. 챔피언 선택 단계에 진입하면 자동으로 데이터가 공유됩니다

### 2. 게스트 (관전하는 사람)

1. 웹 브라우저에서 애플리케이션에 접속합니다
2. 공유받은 **방 코드**를 입력합니다
3. **"참가"** 버튼을 클릭합니다
4. 호스트의 밴픽 화면이 실시간으로 표시됩니다

## ⚔️ 피어리스 제도

### 모드 설명

#### 논 피어리스 (None Fearless)
- 일반 모드
- 챔피언 선택에 제한 없음
- 모든 세트에서 동일한 챔피언 선택 가능

#### 소프트 피어리스 (Soft Fearless)
- **자기 팀**이 이전 세트에서 선택한 챔피언만 선택 불가
- 상대 팀이 선택했던 챔피언은 선택 가능
- 예시: 우리 팀이 1세트에서 아리를 선택 → 2세트에서 우리 팀은 아리 선택 불가, 상대 팀은 가능

#### 하드 피어리스 (Hard Fearless)
- **양 팀 모두**가 이전 세트에서 선택한 챔피언 선택 불가
- 한 번 선택된 챔피언은 경기가 끝날 때까지 양 팀 모두 사용 불가
- 예시: 1세트에서 아리가 선택됨 → 2세트에서 양 팀 모두 아리 선택 불가

### 사용 방법

1. 호스트가 방을 만들면 **피어리스 모드**를 선택할 수 있습니다
2. 챔피언 선택이 완료되면 자동으로 세트 기록에 저장됩니다
3. 제한된 챔피언은 **🚫 아이콘**과 함께 표시됩니다
4. **초기화** 버튼으로 모든 세트 기록을 삭제할 수 있습니다 (확인 필요)

### 화면 표시

- **우리 팀**: 제한된 챔피언은 빨간색 테두리 + 🚫 표시
- **상대 팀**: 제한된 챔피언은 주황색 테두리 + 🚫 표시
- **게임 세트 기록**: 각 세트별 밴/픽 내역을 시각적으로 표시

- **개발 도구**: Claude Sonnet 4.5 (AI-Powered Development)
